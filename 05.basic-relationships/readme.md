# 연관관계 매핑

JPA는 객체의 참조와 테이블의 외래 키를 매핑해서 객체에서는 참조를 사용하고 테이블에서는 외래 키를 사용할 수 있도록 한다.

## 연관관계 매핑 핵심 키워드

- 방향(Direction)
    - 단반향: 한 쪽만 참조하는 관계 (`EntityA --> EntityB`)
    - 양방향: 양쪽 모두 서로 참조하는 관계(`EntityA <--> EntityB`)
- 다중성(Multiplicity)
    - N:1
    - 1:N
    - 1:1
    - N:M
- 연관관계의 주인(owner): 객체를 양방향 연관관계로 만들면 연관관계의 주인을 정해줘야 한다.
